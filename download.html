Install MCPI reborn (our flagship MCPI mod) on ARM (Raspberry Pi, RockPi, etc), Linux, and Windows!

Method A: for armhf/arm64 OSes! (so only for Raspberry Pis/RockPis/etc). Install it on pi-apps: https://github.com/Botspot/pi-apps (Listed as MCPI Modded in the Games section!).

Method B: for x86_64 Linux, get it from our APT repo mcpi-packages by running the following commands:

wget -qO- https://raw.githubusercontent.com/MCPI-Revival/mcpi-packages/master/install.sh | sudo bash -

sudo apt-get install -y minecraft-pi-reborn-client

Method C: for all OSes/architectures! Only use if Method A/B fails or you can't get internet access. Get it straight from the source using debs: https://gitea.thebrokenrail.com/TheBrokenRail/minecraft-pi-reborn/src/branch/master/docs/INSTALL.md. Use two step build for x86_64 Linux and one step build for ARM.

Windows instructions: MCPI-Revival on windows is still experimental, to check the instructions, please join our Discord Server.
